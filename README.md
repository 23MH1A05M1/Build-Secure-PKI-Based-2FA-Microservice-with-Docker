# Build Secure PKI-Based 2FA Microservice with Docker

This repository contains my implementation of the **PKI-based TOTP (Time-based One-Time Password) 2FA microservice**.

The project includes:

- RSA 4096-bit student key pair  
- Public/Private keys stored in PEM format  
- Python scripts for generating keys and signing data  
- TOTP generator and verification  
- Dockerfile for building the microservice  
- Cron job setup for automated tasks
